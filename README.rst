Tools for downloading sports statistics.


Installation
------------

1. Install_ pip::
   
   $ sudo easy_install pip
   
2. Install `sportsstats`:: 
   
   $ pip install sportsstats

Usage
-----

For usage info and help::

   $ stats -h
